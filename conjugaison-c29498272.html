    • Le conditionnel pr√©sent

      Par ProfC le 30 Ao√ªt 2017 √† 11:22

      Rappel : Le conditionnel présent se conjugue comme le futur. Il faut ajouter les terminaisons à l'infinitif. Les terminaisons du conditionnel sont celles de l'imparfait : -ais, -ais, -ait, -ions, -iez, -aient. 

      Le conditionnel permet en général d'exprimer une action qui ne s'est pas encore produite et dont on n'est pas certain qu'elle se produira ( exemple : si je gagne au loto, j'irais en Martinique). On l'utilise aussi pour exprimer un souhait, un désir, un rêve ( exemple : je souhaiterais partir en Martinique). 

      Pour t'entraîner, tu peux faire les exercices suivants : 

      - sur françaisfacile

      - sur bonjourdefrance

      - sur un site d'instituteur

    • Le futur simple

      Par ProfC le 30 Ao√ªt 2017 √† 11:12

      Rappel : Pour conjuguer le futur, il suffit d'ajouter les terminaisons ( -ai, -as, -a, -ons, -ez, -ont) à l'infinitif du verbe. 

      Attention aux verbes suivants : 

      - les verbes qui se terminent par un -e : on enlève le -e final avant d'ajouter la conjugaison ( dire -> je dirai)

      - les verbes courir et mourir : je mourrai, je courrai

      Le futur simple exprime une action qui va se passer de façon certaine dans le futur. Si l'action n'est pas certaine, qu'elle est soumise à une condition, alors il faut utiliser le conditionnel. 

      Pour t'entraîner, tu peux faire les exercices suivants : 

       - sur lingolia

      - sur ortholud ( plusieurs exercices à faire)

      - sur bonjourdefrance

      - sur francaisfacile

    • Comment r√©viser la conjugaison ?

      Par ProfC le 23 Novembre 2016 √† 15:01

      Si vous voulez réviser votre conjugaison, il y a plusieurs façon de faire : 

      - avec un Bescherelle : prenez une page au hasard et essayez de conjuguer le verbe au temps que vous voulez. Ca peut même devenir un jeu : avec un ou plusieurs camarades, posez-vous des "colles" à tour de rôle et comptez les points. 

      - sous forme de bataille navale : chaque joueur choisis plusieurs verbes et tous les joueurs se mettent d'accord sur un même temps. Faites une grille comme pour une bataille navale sauf que pour trouver les bateaux il faut conjuguer le verbe. Demander plus d'explications à votre professeur si nécessaire. 

      - sur Internet : il y a plusieurs sites très bien qui proposent de s'entraîner, notamment Ortholud (http://www.ortholud.com/conjugaison.html ) et logiciels éducatifs ( http://www.logicieleducatif.fr/index_conjugaison_grammaire.php)
